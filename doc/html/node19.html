box2d_wav

Wavelet of the "Box" frame.


Syntax

out = box2d_wav(kx, ky, nang)


Description

Compute the wavelet of the "Box" frame, namely the angular sector which take 1 if pi/2 < = abs(K) < pi, - pi/nang < arg(K) < = pi/nang, with K = (kx, ky), and 0 elsewhere.


Input Data

kx
[MATRIX]: the horizontal frequencies;

ky
[MATRIX]: the vertical frequencies;

nang
[INTEGER]: the number of angular sectors on which the frame decomposition is computed.


Output Data

out
[MATRIX]: the computed function on the frequency plane.
