box2d_app

Approximation function of the "Box" frame


Syntax

out = box2d_app(kx, ky)


Description

Compute the approximation function of the "Box" frame, namely the isotropic function which takes 1 if (kx2 + ky2) < pi/2, and 0 elsewhere.


Input Data

kx
[MATRIX]: the horizontal frequencies;

ky
[MATRIX]: the vertical frequencies;


Output Data

out
[MATRIX]: the computed function on the frequency plane.

Location
