ifwt2d

2D inverse framed wavelet transform


Syntax

out = fwt2d( yastruct, ['import', import] );


Description

fwt2d computes the 2D inverse Framed Wavelet Transform on the basis of the fwt2d's result.


Input Data

yastruct
[STRUCT]: the result of fwt2d

import
[STRING]: can be 'app', 'wav' or 'rem' and allow ifwt2d to rebuild only respectively the approximation, the wavelet coefficient or the high frequency remainder.


Output Data

out
[MATRIX]: the reconstructed image.
