yachirp

Compute a typical chirp cosine


Syntax

[out] = yachirp([n])


Description

This function computes another typical chirp cosine on n points (the default is 256)


Input Data

n
[INTEGER]: the number of points.


Output Data

out
The output chirp.


Example(s) 

>> ch = yachirp(256)
>> plot(ch)
