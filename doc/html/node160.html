yathresh

Hardly threshold a matrix.


Syntax

[out] = yathresh(mat,level,['absolute'])


Description

Return the matrix 'mat' hard thresholded by 'level'. This level can be realtive (in percent of maximum) or absolute.


Input Data

mat
[MATRIX]: the input matrix;

level
[REAL]: the level of thresholding. If relative, level is 0<level<1, with 0 and 1 corresponding respectively to 0% and 1% of thresholding. If absolute, level is considered as a value.


Output Data

out
[MATRIX]: the thresholded matrix.


Example(s) 

>> mat = rand(100,100);
>> nmat = yathresh(mat,0.5);
>> figure; imagesc(mat);
>> figure; imagesc(nmat);
