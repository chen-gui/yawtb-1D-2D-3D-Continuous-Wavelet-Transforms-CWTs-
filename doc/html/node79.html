inputeof

Work like input(msg,'s') but ends by a EOF


Syntax

out = inputeof(msg)


Description


Input Data

msg
[STRING]: your prompt message


Output Data

out
[STRING]: the text entered before the closing EOF


Example(s) 

>> s=inputeof(['Enter something and terminate by typing ...
EOF<enter> on a new line']);
