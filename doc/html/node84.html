list_elem

Return the k-th element of rec if it exists.


Syntax

out = list_elem(rec,k,def)


Description

Return the k-th element of rec if exists. If not, default value def is returned.


Input Data

rec
[LIST] the input list;
k
[INTEGER] the element index;
def
[MISC] default value.


Output Data

out
the k-th element.
