whatin

Return information about a matrix


Syntax

whatin mat whatin(mat)


Description

Return the major information of a matrix mat.


Input Data

mat
[MATRIX|STRING]: the input matrix or its name in the matlab workspace. Note: Don't use a name of variable inside a mfile because 'whatin' looks for the variable in the 'base' workspace.


Example(s) 

>> whatin(rand(5,5) + i*rand(5,5))
