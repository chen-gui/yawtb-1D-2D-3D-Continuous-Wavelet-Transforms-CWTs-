regrep

Perform a substring replacement on string using regular expression.


Syntax

out = regrep(pattern,replace,string)


Description

This program performs the replacement of the substring 'pattern' into 'string' with 'replace'. The writting of the pattern follows the regular expression syntax. Remark: regrep uses the perl executable.


Input Data

pattern
[STRING]: the pattern to search into string;

replace
[STRING]: the replacing word;

string
[STRING]: the string processed.


Output Data

out
[STRING]: the resulting string.


Example(s) 

>> regrep('([a-z])e','a','hello')

Location

tools/misc/regrep.m
