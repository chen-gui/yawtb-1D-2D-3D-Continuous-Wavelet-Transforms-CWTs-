ifftsph

Inverse Spherical Harmonic Transform


Syntax

[out] = ifftsph(fmat, theta, phi)


Description

Compute the inverse Spherical Harmonics transform.


Input Data

fmat
[MATRIX]: The SPherical Harmonics coeffcients, for instance the result of fftsph.

theta, phi
[MATRICES]: the equiangular grid in latitude-longitude coordinate where the function is defined. The size of this grid must be (2M+1)xN with an odd number of latitudes.


Output Data

out
[MATRIX]: the rebuilt function.


Example(s) 

>> %% Creating the equiangular spherical grid
>> [phi, theta] = meshgrid(vect(0,2*pi,32,'open'), vect(0,pi,17));
>> %% Creating a function
>> mat = yaspharm(theta, phi, 3,2) + yaspharm(theta, phi, 5,0);
>> figure; yashow(mat, 'spheric', 'relief', 'mode', 'real');
>> %% Computing the SH transform
>> tmat = fftsph(mat);
>> nmat = ifftsph(tmat, theta, phi);
>> yapsnr(mat,nmat)
>> figure; yashow(nmat, 'spheric', 'relief', 'mode', 'real');
