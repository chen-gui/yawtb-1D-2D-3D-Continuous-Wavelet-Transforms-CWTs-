yapuls

Pulsation vector


Syntax

puls = yapuls(npuls)


Description

yapuls returns a pulsation vector puls of size npuls which is the concatenation of two subvectors whose elements are respectively in [0,) and [- , 0). Useful function when computing wavelets directly in the Fourier domain.


Input Data

npuls
[REAL SCALAR]: length of the pulsation vector


Output Data

puls
[REAL VECTOR]: the pulsation vector


Example(s) 

>> puls5 = yapuls(5)
>> puls6 = yapuls(6)
prints and returns a 5-length, and a 6-length pulsation vector.
