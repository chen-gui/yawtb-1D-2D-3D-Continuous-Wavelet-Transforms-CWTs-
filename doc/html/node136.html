yamse

Mean square error between signals or images


Syntax

mse = yamse(psig, nsig)


Description

This function computes the Mean Square Error (MSE) between a pure and a noisy signal/image. The formula used is

MSE = | psig - nsig|2


Input Data

psig
[REAL VECTOR|REAL MATRIX]: real vector or real matrix giving the base signal, generally the signal without noise
nsig
[REAL VECTOR|REAL MATRIX]: real vector or real matrix giving the noisy signal.


Output Data

mse
[REAL SCALAR]: The Mean Square Error


Example(s) 

>> pmat = theL(256); yashow(pmat,'square');
>> nmat = pmat + randn(size(pmat))/5; yashow(nmat,'square');
>> yamse(pmat, nmat)
