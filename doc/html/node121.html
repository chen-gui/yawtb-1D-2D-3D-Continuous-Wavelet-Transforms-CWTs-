theL

A 2D academic example: the letter L in a NxN matrix


Syntax

out = theL([N])


Description

Return a binary square matrix of NxN size with the letter L inside.


Input Data

N
[INTEGER] the width of the returned square matrix (default 64)


Output Data

out
[BINARY MATRIX] the resulting matrix.


Example(s) 

>> mat = theL;
>> imagesc(theL);
>> colormap(gray);
