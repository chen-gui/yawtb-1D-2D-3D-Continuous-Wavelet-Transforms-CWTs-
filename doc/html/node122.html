thedisk

Create a disk in a square image.


Syntax

[out] = thedisk([n] ['Center', center] [,'Radius', radius] [,'Smooth', smooth])


Description


Input Data

n
[INTEGER]: the size of the image (default 64);

center
[REAL VECTOR]: specify the center of the disk in (x,y) coordinates knowing that the complete image is the domain [-1,1]x[-1,1]. 'center' is set to the origin [0 0] by default.

radius
[REAL]: the radius of the disk relatively to the size of the output matrix (0: nothing, 1: the whole matrix).

smooth
[REAL] specify if the disk must be smooth on the edge.


Output Data

out
[BINARY MATRIX]:


Example(s) 

>> mat= the disk;
>> yashwo(mat);
