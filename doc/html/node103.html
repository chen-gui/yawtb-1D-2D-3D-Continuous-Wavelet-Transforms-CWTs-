pgmwrite

Write an image in the raw PGM format.


Syntax

pgmwrite(mat, fname)


Description

Write an image 'mat' in a file 'fname' in the raw PGM format.


Input Data

mat
[ARRAY]: the origanl matrix. If this file is not in the uint8 format, it is converted.

fname
[STRING]: the string containing the name of the file to write in the current dir.


Output Data

[]:
