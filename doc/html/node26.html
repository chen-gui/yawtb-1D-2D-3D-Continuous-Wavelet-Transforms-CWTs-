cube

Generate a 3D sample cube.


Syntax

[out] = cube(n [,ncube])


Description


Input Data

n
[INTEGER]: The size of the cubic domain where the cube is defined.

ncube
[INTEGER]: the size of the cube set to n/2 by default.


Output Data

out
[ARRAY]: the volume describing the cube (1 inside & 0 outside)


Example(s) 

>> cub = cube(64);
>> yashow(cub);
