the3dL

Generate a the 3D version of the L shape.


Syntax

[out] = cube(n [,ncube])


Description


Input Data

n
[INTEGER]: The size of the cubic domain where the L shape is defined.

ncube
[INTEGER]: the size of the cube containing the L set to n/2 by default.


Output Data

out
[ARRAY]: the volume describing the L shape (1 inside & 0 outside)


Example(s) 

>> theL = the3dL(64);
>> yashow(theL);
