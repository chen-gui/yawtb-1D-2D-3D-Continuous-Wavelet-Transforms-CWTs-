sdog1d

1D Scaling Difference Of Gaussian wavelet in the frequency domain


Syntax

[out] = sdog1d(k,alpha)


Description

This function returns out the 1D Scaling Difference Of Gaussian wavelet in the frequency domain defined in that domain by: 

  out = exp( - k.^2 ) - exp ( - alpha^2 * k.^2)


Input Data

k
[REAL MATRIX]: The frequency. You can use the YAWTB function vect to create it (see example below).

alpha
[REAL SCALAR]: The wavelet parameter.


Output Data

out
[REAL MATRIX]: The wavelet in frequency.


Example(s) 

>> step = 2*pi/128;
>> k = -pi : step : (pi-step);
>> wav = sdog1d(freqs,6);
>> plot(k,wav);

Note that the two first lines are implemented by the YAWTB vect function: 

>> k = vect(-pi,pi,128,'open');
