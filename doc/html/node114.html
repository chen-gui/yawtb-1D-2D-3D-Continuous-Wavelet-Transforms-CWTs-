setyawtbprefs

Set a YAWtb preference


Syntax

setyawtbpref(property_name, value)


Description

Set the property named 'property_name' in YAWtb preference to 'value' This property must exist.


Input Data

property_name
[STRING]: The name of the property
value
[MISC]: The new value
